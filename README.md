# 🎲 Ludo Master - Premium Offline Android Experience

**Ludo Master** is a high-performance, visually stunning Ludo board game designed specifically for the Android platform. Focused on delivering a premium offline experience, the app combines the classic joy of Ludo with modern design aesthetics and smooth, tactile gameplay.

## 🌟 Key Features

- **🏆 Master Branding**: Experience a premium look with our custom-designed 3D board logo and high-end iconography.
- **🤖 Vs Computer Mode**: Challenge our intelligent AI in a competitive battle for the board.
- **👥 Pass N Play**: Local multiplayer experience with friends and family on a single device.
- **✨ Modern Light UI**: A clean, vibrant interface optimized for visibility and engagement.
- **⚡ High Performance**: Native Java for speed, smooth animations, and minimal battery usage.

## 🛠️ Technical Stack

- **Language**: Java (Native Android SDK)
- **UI Framework**: XML Layouts / Material Design
- **Minimum SDK**: 24 (Android 7.0+)
- **Build Tool**: Gradle 8.13

## 📂 Directory Structure
The repository is organized as follows:
- `/app`: The main Android application source code.
- `/marketing_website`: Source code for the [official website](https://ludomaster.ct.ws/?i=1).
- `/gradle`: Build configuration files.

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jdhdhgvdhe/ludomaster.git
   ```
2. Open the project in **Android Studio**.
3. Sync Gradle and click **Run**.
